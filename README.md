# Fleetal — Logistics Operations Dashboard

A comprehensive logistics management system for fleet operations, deliveries, drivers, and customer management.

**Live Demo:** [https://blvklegend.github.io/Fleetal/](https://blvklegend.github.io/Fleetal/)

This admin dashboard connects to the main Fleetal platform. Visit the full site here: [https://alobaiyunade.wixsite.com/fleetal](https://alobaiyunade.wixsite.com/fleetal)

---

## Tech Stack

| Layer              | Technology                          |
|--------------------|-------------------------------------|
| Framework          | React 19 + Vite 8                   |
| Routing            | React Router v7                     |
| Data Fetching      | TanStack Query v5                   |
| State Management   | Zustand v5 (with persistence)       |
| Charts             | Recharts                            |
| Styling            | Tailwind CSS v3                     |
| Icons              | Lucide React                        |
| Storage            | localStorage (demo mode)            |

---

## Modules

| Page         | Key Features |
|--------------|--------------|
| **Dashboard** | Real-time KPIs, interactive charts, activity feed, and role-specific insights |
| **Deliveries** | Full CRUD operations, smart filters, search, pagination, and detail views |
| **Drivers**   | Driver profiles, performance tracking, ratings, and status management |
| **Fleet**     | Vehicle health monitoring, fuel levels, service scheduling |
| **Customers** | Customer segmentation, spend tracking, and contact management |
| **Analytics** | Revenue trends, on-time performance, volume analysis, and forecasting |
| **Settings**  | Theme settings, role simulation, workspace controls, and data management |

---

## Key Features

- **Role-Based Access Control** — Supports Admin, Dispatcher, Driver, and Viewer roles with proper data isolation and permission handling.
- **Workspace Mode** — Easily switch between demo data and a clean workspace.
- **Command Palette** — Quick keyboard navigation (`Cmd/Ctrl + K`).
- **Responsive Design** — Works smoothly on desktop and mobile with slide-in sidebar and adaptive modals.
- **Dark Mode** — Fully persisted dark/light theme.
- **Polished UX** — Loading skeletons, toast notifications, confirmation dialogs, and smooth interactions.
